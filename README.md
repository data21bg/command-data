## 第1章  数据科学就是OSEMN 

数据科学：(1)数据获取；(2)数据清洗；(3)数据探索；(4)数据建模；(5)数据解释。  
这5个步骤一起构成了OSEMN（发音同awesome）模型。  
  
1. 数据获取
* 从其他地方（如网页或服务器）下载数据
* 从数据库或API（如mysql或Twitter）中查询数据
* 从其他文件（如html文件或电子表格）中提取数据
* 自已生成数据（如读取传感器或进行调查）
2. 数据清洗
* 常见的操作：
* 行过滤
* 列抽取
* 值替换
* 单词提取
* 缺失值处理
* 数据格式转换
  
**任何数据项目中80%的工作都是数据清洗**

3. 数据探索
* 查看数据
* 从数据中推导统计量
* 创建有趣的可视化
4. 数据建模
* 为数据建立一个统计模型
* 包知聚类、分类、回归以及降维
5. 数据解释
* 从数据中得出结论
* 评估结查的含义
* 告知你的结果

命令行的5个重要优势:  
> 活性、可增强、可扩展、可扩充、无处不在
